Python package notes
--------------------

* pyqtgraph. A bit faster for interactive use than the excellent, (but
  sometimes sluggish) matplotlib.
  http://www.pyqtgraph.org/

* python-hdf4. A fork of pyhdf for hdf4 handling (python 3 compatible) 
  http://fhs.github.io/python-hdf4/
